How to build this in Google Forms

  1. Go to forms.google.com → Blank form
  2. Copy each section below as a Section in the form
  3. Star (*) = mark as Required
  4. Scale questions = Linear scale (1–5)
  5. Multi-select = Checkboxes
  6. Open text = Paragraph
  7. Go to Responses tab → green Sheets icon → Create a new spreadsheet
  8. Share the form link with the same people who have GitHub access
